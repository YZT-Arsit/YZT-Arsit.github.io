## Selected Highlights

### Agent Benchmark and Recovery Framework

- Built a tool-calling benchmark with **24 hierarchical tasks** and automatic validation.
- Developed an adaptive **Plan-then-Act agent** for noisy execution settings.
- Achieved **88% success rate** versus **25%** for a ReAct baseline.
- Reduced average tool calls from **42.88** to **2.67**.

### Enterprise Email Risk Modeling

- Processed **24,000 unlabeled enterprise email logs** with weak supervision and active learning.
- Defined a practical **5-class risk taxonomy** for downstream security tasks.
- Improved domain performance with MLM pre-training and fine-tuning on `BERT-Base-Multilingual-Cased`.

### Legal Document RAG System

- Built a RAG pipeline for **12,000+ Chinese legal documents**.
- Proposed a recursive aggregation strategy based on table-of-contents extraction.
- Improved evaluation results by **+0.25 BLEU-4** and **+0.30 ROUGE-L** on **2,000** samples.
